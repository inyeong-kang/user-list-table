# 회원 목록 관리 테이블

## 프로젝트 세팅 방법

**패키지 설치**

```
npm i
```

**프로젝트 개발 서버 실행**

```
npm run dev
```

-   [http://localhost:5173/](http://localhost:5173/) 에 접속하여 구현 결과를 확인할 수 있습니다.

**스토리북 실행**

```
npm run storybook
```

-   [http://localhost:6006/](http://localhost:6006/) 에 접속하여 구현 결과를 확인할 수 있습니다.
